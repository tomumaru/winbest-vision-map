# 3D思考空間ビジュアル

ChatGPTの会話履歴から抽出したテーマを、3D空間上のノードとして配置するための初期プロトタイプです。

## 3軸の考え方

最初の設計では、誰にでも当てはまる汎用軸をそのまま表示するのではなく、汎用的に測れる構造にあなた固有の呼び名を与えます。

- X: 具象 ↔ 構造
- Y: 内省 ↔ 接続
- Z: 定着 ↔ 流動

「AIスラップ」「UUID」「流動性」のような言葉は、軸そのものというより、あなたの思考空間に現れる特徴的なクラスタ名やノード名として扱うのが自然です。

## 操作

- 表示モード: Winbest、思考、企画、状態、物語を切り替え
- Axes: 軸説明を詳細 / 短縮 / 非表示で切り替え
- ドラッグ: 視点回転
- ホイール: ズーム微調整
- Ctrl + ホイール: 段階ズーム
- クリック: ノード選択
- 軸ラベルのダブルクリック / ダブルタップ: その軸が見やすい角度へ補正
- スマホのスワイプ: 視点回転
- スマホのタップ: ノード選択
- スマホのピンチイン / ピンチアウト: 段階ズーム
- カーソルキー: 画面上で隣のノードへ移動

## 次の入力データ

実際のChatGPT履歴を使う場合は、会話エクスポートから次を抽出します。

- テーマ
- 代表フレーズ
- 感情や熱量
- 抽象度
- 他テーマとの参照関係
- 時系列での変化

## PWAとして使う

このアプリはPWA対応済みです。GitHub PagesなどHTTPSで配信されるURLをスマホで開き、ブラウザのメニューからホーム画面に追加すると、以後はアプリのように起動できます。

- Android Chrome: メニューから「ホーム画面に追加」
- iPhone Safari: 共有ボタンから「ホーム画面に追加」

一度読み込むと、主要ファイルはオフラインキャッシュされます。
